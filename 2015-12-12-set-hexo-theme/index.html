更换hexo主题

发表于 2015-12-12 书写 1,086 字   |   分类于 hexo > hexo美化   |   访问 次   |  

本文旨在帮助你为自己的 hexo 博客更换主题。

1. 阅读需知

  • 运行环境： Win8.1 + Hexo3.1.1
  • 预备基础：搭建 Hexo 博客（方法参见 在Github上搭建Hexo博客 ）。

2. 选择心仪的Hexo主题

  1. 如果你无处下手，可以到下面2个网址中淘选：
    • 比较全的Hexo主题列表： Themes · hexojs/hexo Wiki · GitHub
    • 知乎上的总结： 有哪些好看的 Hexo 主题？ - GitHub - 知乎
  2. 在浏览别人的Hexo博客时发现心仪主题，一般下拉到页面底部，就可以看到博客使用的什么主题，并可点击进入主题下载页面。
