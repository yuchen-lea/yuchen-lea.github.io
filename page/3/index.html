给hexo添加RSS

发表于 2016-01-18 书写 845 字   |   分类于 hexo > hexo设置   |   访问 次   |  

1. 安装RSS插件到本地

<!– more –>

2. 设置全站配置文件 _config.yml

  1. 启用插件：

    plugins:
        - hexo-generator-feed

  2. 设置插件：

    feed:
      type: atom
      path: atom.xml
      limit: 0
      hub:

    参数说明：

    type - Feed type. (atom/rss2)
    path - Feed path. (Default: atom.xml/rss2.xml)
    limit - Maximum number of posts in the feed (Use 0 or false to show all posts)
    hub - URL of the PubSubHubbub hubs (Leave it empty if you don't use it)

  3. 修改 url: 为博客站点地址 默认为 http://yoursite.com/ ，但是为了RSS地址的正确引用，应将其改为自己的博客地址，例如我的配置文件为：

    url: http://yuchen-lea.gitcafe.io/

3. 设置主题配置文件 _config.yml

  1. 如果主题设置中内置有RSS： 只要设置其地址与全站设置文件中的一致即可：

    rss: /atom.xml

  2. 如果主题设置中没有内置RSS： 在完成上述步骤后，还需自行为站点添加RSS链接。更改站点的布局需要修改模版文件，一般在 layout 目录下。如果要在header加入RSS链接，找到 header.ejs 文件，在相应位置添加代码：

4. 检查 RSS 文件的正确性

  1. 执行 hexo g ，在 public 目录应生成 atom.xml
  2. 执行 hexo s , 查看页面的RSS收取点能否打开，文章链接是否正确

  • <2016-01-22 五> 增加 检查 RSS 文件的正确性 部分
  • <2016-01-18 一> 拟定初稿

    • 发音 在あいうえお这5个单元音中，嘴型是张开最大的。
  2. i
    • 平假名 い 片假名 イ

    • 笔顺

    • 字源

    • 发音 不咧嘴，口型非常小。
  3. u
    • 平假名 う 片假名 ウ

    • 笔顺

    • 字源

    • 发音 汉语中“乌”是圆唇音，但是日语的 u 不是圆唇音。几乎不张嘴，口型非常小，牙齿嘴唇只是微微张开一条缝。

    • 平假名 え 片假名 エ

    • 平假名 お 片假名 オ
    • 笔顺
    • 字源
    • 发音 在あいうえお这5个单元音中，唯一的圆唇音。

日语的声调

发表于 2016-01-15 书写 2,462 字   |   分类于 日语 > 日语口语   |   访问 次   |  

本文旨在介绍日语的声调表示方法、类型与发音规律，并附有相关音频以供参考。

本文含有一些特殊字符，目前无法正常显示，正在努力解决中~

日语口语要想发音正确，掌握日语的声调十分重要。

1. 关于声调的基本常识1

在日语中声调称为 アクセント ，背过五十音图的同学可以自己拼读试试（点此查看五十音图的总结），就是英语单词 accent 的音译。不过日语的声调其实更接近汉语的声调，而非英语的声调（强弱重音）。声调广义地可以分为2种：

  • 高低重音：以声音的高低不同表现重音，例如日语、汉语
  • 强弱重音：例如英语

所以日语被称为中国人最容易学习的外语（没有之一）并非信口开河。日语的声调比之汉语更简单：

  • 汉语声调（5种） 阴平 ā，阳平 á，上声 ǎ，去声 à，轻声 a
  • 日语声调（2种） 高（类似阴平/一声），低（类似轻声）

<!– more –>

由于日语发音短促，每个字只有高低两种变化，不能像汉语在一个音节中实现高低变化（比如说“哪”这样的三声）。一般所讲的声调，并非指一个假名的高或低，而是指一个词语中，构成词语的所有字的特定高低顺序。

表示声调高低的方法通常有两种：

  1. 图示法

    直观易懂，常见的图示有2种：

    1. 划线图示：横线在上是高音，横线在下是低音。多用于日语教材，在词汇、句子中直接划线标明高音低音。

      该图即表示第一个音节 ぉ 是低音，二三音节 して 是高音，第四个音节 も 是低音。

    2. 圆点图示：与划线图示的本质相同。一个圆点表示一个音节；圆点在上是高音，圆点在下是低音。与划线图示相比，即便没有词汇实例，也能直观地反映出声调。

      例如上例用圆点图示则表示为：

  2. 标号法

    书写方便，通常表示为⓪◎、①、②、③、④、⑤……，圈内的数字表示：1) 音调从哪个音节由高转低，2) 音调核所在的音节。与图示法的对应关系见下图：

2. 声调示例

以下5段示例皆截取自 万门大学|五十音图第一行 ,仅供学习参考。在此感谢苏曼老师清晰的讲解。

以あ行的5个单元音 あ い う え お 为例：

  • ①调
  • ②调
  • ③调
  • ④调
  • ⑤调
  • ⓪◎调

从录音听，最后2种 ⑤调和⓪◎调 是一样的，因为这里只是以五个音节为例，如果后面再接助词，⑤调的助词是低音，而⓪◎调的助词是高音。通过上文的图示可以更好地理解：只有五个音节时，◎调和⑤调是一样的，只有在多于5个音节时才存在差异。

3. 规律

  1. 前2拍（即前2个假名）的声调相反
  2. 单峰型，没有多次起伏。1个单词中，最多下降一次，而且下降之后不再上升
  3. 标号法中圈内的数字为几，表示从第几个音节开始下降

而要正确发音，我们需要知道2点：

  1. 起始假名是低音还是高音
  2. 从第几个音节开始升降

这2点都可以通过总结的3条规律轻松推得：

类型 起始音节 升降变化
①型 高音 从第二拍下降，一直到最后也不上升
其他 低音 从第二拍升高，一直保持到音调核数字的拍节后下降，然后到最后也不上升

根据上图来记忆会更有效！

4. 实际学习中遇到的问题

  1. 一个单词标明2个音调核：说明该词有2种读法，一般读前面标记，但是读后面的标记亦可。
  2. 一个单词标明2个音调核，中间有“+”号连接：说明该词由2个单词复合而成，这时会出现双峰型。
  3. 说话时带有各种感情，声调会发生变化。在正常情况下，能正常的发出读音就可以了。
